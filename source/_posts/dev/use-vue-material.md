---
[Vue Material](http://vuematerial.io)

## インストール
```
$ yarn add -D vue-material
```

## 使用準備
```
import Vue from 'vue'
import VueMaterial from 'vue-material'
import 'vue-material/dist/vue-material.css'

Vue.use(VueMaterial)
```
